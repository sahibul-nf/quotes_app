## Run Locally

Clone the project

```bash
  git clone https://github.com/sahibul-nf/quran_app
```

Go to the project directory

```bash
  cd quran_app
```

Install dependencies

```bash
  flutter clean && pub get
```

Run the app

```bash
  flutter run
```
